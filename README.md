MSH Rust - Micha1207's Shell Program written in Rust Language.

---

## Getting started
This project is written in **Rust**.

### Features
- Built-in commands: MSH Rust comes with minimal: cat, cd, clear, echo, ls, mkdir, pwd, rm, touch.
- Run any command from $PATH or working directory,
- Colorful prompt,
- Doesn't save history and doesn't support completion

---

### Requirements
Make sure to have [Rust Environment](https://rustup.rs/) installed on your system.

---

1. **Download sources**
   Clone project's repo with `git`:
   ```bash
   git clone https://github.com/Micha1207/msh-rs.git
   cd msh-rs
   ```

   Or download ZIP file directly from [project's GitHub page](https://github.com/Micha1207/msh-rs) and extract it.

2. **Compilation**:
   To compile this project, run `cargo run` in project directory.
